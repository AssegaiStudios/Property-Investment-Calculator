# Security Policy

## Supported Versions

The latest published release is the supported version.

## Reporting a Security Issue

If you identify a security concern, privacy concern, unsafe workbook behaviour, or potentially harmful file issue, please raise it privately with Assegai Studios where possible, or open a GitHub issue if no private contact method is available.

Please do not include sensitive personal or financial information in public issues.

## Spreadsheet Safety

Users should review downloaded files before use and should only enable macros if they understand and trust the source.

This project should not require users to enter sensitive personal identifiers, banking credentials, passwords, or account access details.
