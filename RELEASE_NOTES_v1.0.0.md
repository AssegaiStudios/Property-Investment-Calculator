# Property Investment Calculator v1.0.0

First public release of the Property Investment Calculator by Assegai Studios.

## Overview

This Excel-based calculator helps users estimate mortgage payments, rental income, property cash flow, and investment performance across multiple countries.

## Highlights

- Mortgage payment estimates
- Rental income modelling
- Monthly and annual cash flow analysis
- Property investment summary metrics
- Country-specific rules and assumptions
- Clear blue-cell input approach
- Suitable for property purchase and rental assessment scenarios

## Important

This calculator is provided for educational and illustrative purposes only. It does not constitute financial, mortgage, legal, tax, property, or investment advice.

Users should verify all assumptions and seek appropriate professional advice before making property, lending, tax, or investment decisions.
