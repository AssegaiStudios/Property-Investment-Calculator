# Property Investment Calculator

Multi-country property investment calculator for mortgage planning, rental analysis, cash flow forecasting, and investment evaluation.

Created by **Assegai Studios**.

## Overview

The Property Investment Calculator is an Excel-based workbook designed to help users estimate the financial impact of purchasing a property, either as a home or as a rental investment.

The calculator supports mortgage repayment modelling, rental income analysis, ongoing cost assumptions, country-specific rules, and cash flow forecasting.

## Key Features

- Mortgage payment estimates
- Purchase affordability calculations
- Rental income projections
- Monthly and annual cash flow analysis
- Property investment return indicators
- Country-specific tax, fee, and cost assumptions
- Multi-country support
- Input-led model using clearly marked blue cells
- Summary outputs for comparing affordability and investment performance

## How to Use

1. Open the Excel workbook.
2. Complete all relevant **blue input cells**.
3. Enter or review the relevant **country rules**.
4. Add property, mortgage, rental income, and cost assumptions.
5. Review the calculated summary outputs.

Only edit the blue input cells. Other cells may contain formulas and should not be overwritten.

## Country Rules

The workbook is designed to work across multiple countries. Users should enter or review the relevant local assumptions before relying on the results, including:

- Currency
- Property purchase taxes
- Legal and transaction fees
- Mortgage assumptions
- Rental rules
- Local ownership costs
- Any country-specific charges or tax assumptions

Users are responsible for ensuring country-specific rules are accurate and up to date.
