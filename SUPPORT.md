# Support

## Getting Help

For help with the Property Investment Calculator, please use the GitHub Issues section of this repository.

You can raise an issue for:

- Calculation questions
- Documentation improvements
- Country rule suggestions
- Possible bugs
- Feature requests

## Before Raising an Issue

Please check:

- You have completed all relevant blue input cells
- You have reviewed the country rules
- You have not overwritten formula cells
- You are using the latest release
