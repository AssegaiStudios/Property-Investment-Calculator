# Contributing

Thank you for your interest in improving the Property Investment Calculator.

## How to Contribute

You can contribute by:

- Reporting calculation issues
- Suggesting new country rules
- Improving documentation
- Suggesting new features
- Identifying unclear inputs or outputs
- Testing the workbook with different property scenarios

## Reporting Issues

When raising an issue, please include:

- The workbook version
- The country or jurisdiction being modelled
- A description of the issue
- The inputs used, where relevant
- The expected result
- The actual result

Please do not share sensitive personal, financial, or identifying information.

## Suggested Improvements

When suggesting new functionality, please explain:

- The use case
- Why it would be useful
- Any country-specific assumptions involved
- Any relevant calculation logic

## Professional Advice

Please note that contributions should not be presented as financial, mortgage, tax, legal, or investment advice.
